# Homebrew Formula for Runix CLI

Zero-config cloud deployment platform. Push your code, we handle the rest.

## Install

```bash
brew install bumahkib7/runix/runix
```

Or:

```bash
brew tap bumahkib7/runix
brew install runix
```

## Usage

```bash
runix login
runix deploy
runix status
runix logs my-app
runix rollback my-app
```
